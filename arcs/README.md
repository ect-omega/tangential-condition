# Arcs

This is the main reading surface of the repository.

Each arc folder should be readable on its own and should present prose in a clear local order.

Currently published:

- [Lia](lia/README.md)

Recommended shape:

```text
arcs/
  lia/
    README.md
    01-ground-state.md
    02-pressure.md
  ava/
    README.md
    01-threshold.md
```

Guidelines:

- organize by character-centric thread, not by canon node
- use numbered section files only for local reading order inside an arc
- keep section length comfortable for GitHub reading
- let `reading-orders/` explain how arcs connect to each other
- let convergence happen across arcs without collapsing them into a single mixed arc
