# Reading Orders

This directory exists because the story is nonlinear.

Use it to guide readers through the available material without forcing a single definitive order.

Recommended files:

- `start-here.md`
- `core-path.md`
- `arc-first.md`
- `spoiler-light.md`

Each guide should explain:

- who the order is for
- what kind of experience it creates
- which arcs or sections it expects the reader to open
