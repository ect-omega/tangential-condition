# Editorial Metadata

This directory is not part of the reader-facing surface.

Use it to keep the minimum internal metadata required to maintain the published text responsibly.

Typical contents:

- source maps from public sections to canon nodes
- inbox ingress for polished interpretation drops
- extracted indexes for characters, events, and devices
- canon ref pins
- bridge notes
- merge notes
- UV Layer 1 contracts when relevant

Keep this layer thin. Canon truth still lives in the separate canon repo.
