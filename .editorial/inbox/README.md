# Inbox

This folder is the ingress point for polished interpretation prose that has not yet been published.

Contents are gitignored by default so temporary drops and intermediate analysis do not clutter repository history.

Expected contents:

- polished prose candidates
- intake manifests
- temporary classification notes

Use this folder to stage prose before it is routed into:

- `arcs/`
- `reading-orders/`
- `editions/`
- `.editorial/indexes/`
